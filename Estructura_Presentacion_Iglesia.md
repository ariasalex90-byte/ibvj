## 🎨 GUÍA DE ESTILO PARA EL DISEÑADOR

### Colores Principales:
- Verde Lima/Chartreuse (#A4C639 o similar)
- Blanco (#FFFFFF)
- Azul (para secciones administrativas)
- Tonos cálidos (anaranjado/dorado para ocasiones especiales)

### Tipografía Sugerida:
- Títulos: Sans-serif Bold (fuerte, legible)
- Body: Sans-serif Regular (accesible, clara)
- Énfasis: Serif (para versículos bíblicos)

### Layout Sugerido:
1. **Hero Section:** Portada "Anuncios para Recordar"
2. **Cards Grid:** Peticiones de oración, Ministerios
3. **Tablas:** Programación de oración (responsive)
4. **Secciones destacadas:** Eventos (Hombradía, Campamento, Maratón)
5. **Footer:** Redes sociales con QR o links directos
6. **Sidebar/Modal:** Contactos de líderes

### Elementos Visuales:
- Iconografía religiosa (manos orando, corazones, familia, etc.)
- Imágenes de fondo para atmosfera
- Separadores decorativos (líneas de diamante como en los originales)
- Posibilidad de pestañas para diferentes ministerios
